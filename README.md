# ðŸ‡³ðŸ‡± Dutch Minimum Wage Tracker

This application tracks and visualizes the statutory minimum wage in the Netherlands from 2002 to 2026+.

## Features
- **Historical Data:** View minimum wage trends from 2002 onwards.
- **Inflation Adjustment:** Calculate "Real Wage" using CPI and CAO indices from Statistics Netherlands (CBS).
- **Policy Insights:** Visualize the impact of key policy changes like the 2024 hourly wage introduction.
- **Calculator:** Interactive tool to calculate wage growth between specific dates.

## Data Sources
- **Rijksoverheid:** Statutory minimum wage figures.
- **CBS (Statistics Netherlands):** Inflation (CPI) and Collective Labour Agreement (CAO) wage indices.
